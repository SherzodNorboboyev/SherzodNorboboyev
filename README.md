## 🔭 Some Projects

### Length-Controlled Abstractive Summarization with BART

Academic NLP project exploring length control in abstractive summarization.

- Compared soft prompt/control tokens with hard decoding caps
- Evaluated BART-based summarization on CNN/DailyMail validation samples
- Used ROUGE, BARTScore, Length Accuracy, paired t-test, and bootstrap resampling
- Studied the trade-off between controllability and summary quality

Repository: [length_control_summarization](https://github.com/SherzodNorboboyev/length_control_summarization)

---

### Legal GraphRAG Pipeline — MVP / Work in Progress

Prototype project for legal document retrieval using graph and vector-based search.

- Experiments with legal document ingestion and structured representation
- Uses Neo4j-style graph modeling concepts
- Explores embeddings, semantic chunks, and hybrid retrieval
- Built as a learning and portfolio project, not as a production legal advice system

Repository: [legal-graphrag-pipeline](https://github.com/SherzodNorboboyev/legal-graphrag-pipeline)

---

### ASR Performance Under Noise and Denoising — Work in Progress

Speech-processing learning project focused on ASR robustness.

- Experiments with noisy speech inputs
- Studies how noise and denoising affect transcription quality
- Uses WER and CER-style evaluation ideas
- Intended as a learning project in speech processing and ASR evaluation

Repository: [asr_noise_reduction_project](https://github.com/SherzodNorboboyev/asr_noise_reduction_project)
